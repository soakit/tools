# 语音识别对比报告

对比文件：
- 音频：123.第123集音频.mp3
- 文件A：123.第123集音频.srt
- 文件B：123.第123集音频1.srt

## 结论：文件A（123.第123集音频.srt）的文字识别明显更准确

这是一段评书（讲述姚广孝/道衍和尚与徐辉祖的历史故事）。通过关键专有名词和成语逐处比对，
文件A 在几乎所有关键名词、人名、成语上都识别正确，而文件B 大量出错。

## 关键对比清单（A: 123.第123集音频.srt ✅ / B: 123.第123集音频1.srt ❌）

### 人名、地名、职衔（最致命的错处）
| 正确/文件A | 文件B 错误 |
|---|---|
| 道衍和尚 / 姚广孝 | 道演和盛、饶廣孝 |
| 世子朱高炽 | 猪高痴 |
| 燕王朱棣 | 都地 / 猪弟 / 阉王 |
| 大谋士袁珙 | 渊虎 |
| 老将军顾成 | 不成 |
| 大将薛禄 | 雪路 |
| 金川门 | 京村门 / 京春门 |

| 魏国公徐辉祖 | 徐徽祖（此处两者相当） |

### 通篇同音错字模式
文件B 存在系统性的同音/近音替换问题，例如：
- 和尚庙 → 合圣庙
- 挂单 → 挂大头/挂丹
- 踢枪 → 踢车
几乎每隔一两句就有一个明显错误。

## 两个次要维度

### 时间轴精度
- 文件A：时间戳更细密、贴合断句（如 00:00:04,459 --> 00:00:06,040 拆得很碎），真正按语音对齐。
- 文件B：每条固定 5 秒一跳（04,459→09,460→14,460...），是等距分段而非真正按语音对齐，做字幕会不同步。

### 标点
- 文件A：几乎无标点。
- 文件B：有逗号、句号，排版更像句子。但这是它唯一的"好看"之处，内容错误太多。

## 总评

| 维度 | 文件A | 文件B |
|---|---|---|
| 人名地名准确度 | ✅ 大部分正确 | ❌ 大面积错误 |
| 成语准确度 | ✅ 准确 | ❌ 多处错误 |
| 时间轴对齐 | ✅ 真实语音对齐 | ❌ 等距 5 秒分段 |
| 标点 | ❌ 几乎无 | ✅ 有逗号句号 |

## 最终建议

推荐使用 文件A（123.第123集音频.srt），它才是真正"听得懂评书"的识别结果，
文件B 基本是同音字错误集锦。

如需更干净的成品字幕，可基于 文件A 补上标点、做一次校对润色后输出。
