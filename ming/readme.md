# 大明王朝

根据著名历史作品《明朝那些事儿》改编的国产历史动画。从朱元璋出生讲起，到崇祯皇帝自缢、明朝灭亡，以史料为基础，以年代和具体人物为主线，以动画评书的形式，对明朝十七帝及王公权贵、小人物的命运进行全景展示，尤其侧重官场政治、战争、帝王心术，并加入对当时政治经济制度、人伦道德的演义。
